Prévisualisation du Favicon

Voici comment le favicon apparaîtra :

16x16px: (Taille standard dans les onglets)
32x32px: (Taille dans les favoris)
64x64px: (Taille agrandie pour voir les détails)
128x128px: (Taille sur mobile/appareils)
Note : Le favicon sera affiché en rouge (#d41e24) avec le texte "RED-K" en blanc. Il apparaîtra dans l'onglet du navigateur, les favoris, et sur les appareils mobiles.
