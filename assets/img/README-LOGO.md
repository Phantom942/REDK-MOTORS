# Logo RED-K MOTORS - Guide d'utilisation

## Fichiers disponibles

### logo-neon.svg
Version complète avec effet néon multi-couches et fond sombre. Idéal pour :
- Utilisation sur fond sombre
- Réseaux sociaux avec thème sombre
- Présentations et documents

### logo-neon-simple.svg
Version simplifiée avec fond transparent. Idéal pour :
- Utilisation sur différents fonds
- Intégration dans d'autres designs
- Export en PNG avec fond personnalisé

## Conversion SVG vers PNG

Pour convertir le SVG en PNG, vous pouvez utiliser :
- **En ligne** : https://convertio.co/svg-png/
- **Adobe Illustrator** : Ouvrir SVG → Exporter → PNG
- **Inkscape** (gratuit) : Ouvrir SVG → Fichier → Exporter en PNG

## Personnalisation

Les fichiers SVG peuvent être modifiés avec n'importe quel éditeur de texte ou logiciel vectoriel :
- Adobe Illustrator
- Inkscape (gratuit)
- Figma
- Sketch
